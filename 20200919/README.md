## SSMS 下載 ( Free )
* Google "SSMS Download"
* 注意版本
* Only for Windows ( 其他作業系統可使用 Azure Data Studio )
* Update ( zh-tw →  en-us )

## 連線至伺服器
* 利用 **[Del]** 鍵刪除歷史清單
* 連接 Azure Storage 可**查看**Blob Container


## 檢視
* 物件總管 [F8]
* 物件總管詳細資料 [F7]
* 已註冊的伺服器 [Ctrl+Alt+G]
* 範本瀏覽器 [Ctrl+Alt+T]
